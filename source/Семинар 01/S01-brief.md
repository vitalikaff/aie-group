# Семинар S01. Среда, Git, шаблонный репозиторий, NumPy

## Связь с курсом

Первый семинар курса «Инженерия Искусственного Интеллекта».  
Опирается на материалы Лекции 01 (обзор курса, роли, жизненный цикл ИИ-систем).

Семинар задаёт базу для всех последующих занятий:

- рабочая среда для курса;
- единая структура студенческого репозитория;
- базовые навыки работы с Git;
- минимально необходимый уровень владения NumPy.

## Цели семинара

После семинара и выполнения домашнего задания студент:

- умеет настроить рабочую среду под курс (Python, IDE/Jupyter, необходимые пакеты);
- понимает принцип работы Git и может сделать базовый цикл: clone → status → add → commit → push;
- понимает структуру и назначение шаблонного репозитория `aie-student-template`;
- умеет создавать и изменять массивы NumPy и выполнять над ними базовые операции.

## Краткое содержание

1. Вступление, связь с лекцией и целями курса.  
2. Настройка среды:
   - проверка версии Python;
   - установка/проверка необходимых инструментов;
   - запуск IDE / Jupyter.
3. Введение в Git:
   - что такое репозиторий;
   - локальный и удалённый репозиторий;
   - базовые команды.
4. Обзор шаблонного репозитория `aie-student-template`:
   - структура каталогов;
   - папка для семинаров;
   - папка для итогового проекта;
   - ключевые файлы.
5. Базовые операции в NumPy:
   - создание массивов;
   - форма (shape), типы (dtype);
   - индексирование и срезы;
   - векторные операции и базовые статистики.

## Формат проведения

- Формат: демонстрация и подробный разбор кода преподавателем (live-coding) с краткими вопросами к аудитории.
- Студенты на семинаре в первую очередь слушают и смотрят, домашнее повторение и оформление выполняют самостоятельно после занятия.

## Результат для студента

К концу семинара студент:

- понимает, как он будет работать с материалами курса через Git и GitHub;
- знает, где в его репозитории будут лежать домашние задания и проект;
- видел «эталонный» пример работы с NumPy, который сможет воспроизвести дома.

После выполнения домашнего задания у студента:

- создан публичный репозиторий на GitHub на основе `aie-student-template`;
- настроен локальный доступ к репозиторию через Git;
- в репозитории есть папка `homeworks/hw01/` с первым ноутбуком/скриптом по NumPy и тестовым коммитом.
